# 1. Number type

- int 정수형
- float 부동소수형
- complex 복소수형

<br>

## int 정수형
- 소수가 아닌 양/음 모든 숫자
```pythona 
a = 1
b = 3243200001019
c = -32423
```

## float 부동소수형
- 소수점을 포함한 숫자
```pythona 
a = 1.0
b = -2.45345
c = 4.55e10
```

## complex 복소수형
- 복소수를 표현한 자료형
```python
a = 1 + 2j
```
<br>

# 2. String 
- 문자, 단어 등으로 구성된 문자들의 집합
- 인덱싱이 가능하다.
```python
a = "123"
b = "string string string"
c = "c"
```
<br>

# 3. Bool
- 참(True)과 거짓(False)을 나타내는 자료형이다.

```python
a = True
b = False
```
